# Bookmarks-manager

A no account, free, transparent Bookmarks manager to use on browser app open source

# Features

- Webapp
- Mobile first
- Uses LocalStorage
- No account needed
- Open source code
- Grid of bookmarks
- Search
- Import/export
- P2P Sync
- Sort by name and by most clicks
- No dependencies besides peers.js

# Optional

- Custom Background
- Folder system
- Drag order of bookmarks

# Settings

- Sync
- Export/Import
- Order
- Theme
